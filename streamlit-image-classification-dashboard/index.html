Streamlit computer vision portfolio project

AI Image Classification Dashboard

A deploy-ready Streamlit application that classifies uploaded images with Hugging Face computer vision models, tracks prediction history, visualizes confidence, compares model performance, and exports classification results.

Applied AI deployment skills

01

Loads Hugging Face image-classification pipelines with Streamlit resource caching.

02

03

Stores in-session classification history with image previews and model metadata.

04

Builds analytics charts and downloadable CSV results for professional reporting.

Live app note

Streamlit apps deploy separately from this static Vercel portfolio.

Vercel serves this project page as a static portfolio artifact. The interactive Python app runs with Streamlit and should be deployed to Streamlit Community Cloud using streamlit_app.py as the entry point.

Main file: streamlit_app.py Dependencies: requirements.txt Target host: Streamlit Community Cloud
