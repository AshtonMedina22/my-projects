Main capstone project

Trip Planner AI Agent with live travel data.

A production-style travel agent with a live Vercel app and a Streamlit implementation. It calls OpenStreetMap tools, retrieves Wikivoyage context, validates itinerary JSON, renders maps, persists state, and learns from user feedback.

Open live app Project brief
2 Agent tools: POI search and guide retrieval
OSM Live geocoding and POI data
Map PyDeck route visualization

Architecture

Agentic planning with reliability controls.

01

OpenAI Responses API tool loop with strict schemas and max-step limits.

02

Nominatim and Overpass requests include User-Agent headers, retries, timeouts, and caching.

03

Wikivoyage RAG chunks are retrieved with TF-IDF ranking and source metadata.

04

Generated itinerary POI IDs are validated against actual tool-returned records.
