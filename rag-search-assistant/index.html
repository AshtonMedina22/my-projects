Streamlit RAG app

RAG Search Assistant

A deploy-ready Streamlit prototype that uploads text documents, chunks them with overlap, retrieves source passages, assembles a RAG prompt, and estimates routing/cache cost savings.

Build path

Ready for Streamlit Cloud now, upgradeable later.

01

Runs without secrets so visitors can upload text files and test retrieval immediately.

02

Shows chunk previews, source URLs, top similarity scores, raw retrieval output, and assembled prompt context.

03

Includes a cost-control view for routing traffic between rule-based answers, cheaper models, and premium models.
