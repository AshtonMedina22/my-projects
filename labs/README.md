# Labs

This folder contains notebook-based coursework, Jupyter exports, and exploratory lab artifacts.

The public portfolio is the Next.js app at the repo root:

- `app/` - App Router pages and live project routes
- `app/api/` - serverless APIs for live demos
- `lib/` - project registry and shared runtime code

Use `labs/` for reference material and course evidence that should remain available in Git without cluttering the production app surface.

## Structure

- `imported-codecademy/` - root-level Codecademy notebook exports moved out of the repo root.
- `engineer-neural-networks-portfolio/` - Banking77 transformer notebook evidence.
- `frankenstein-finetuning-portfolio/` - Frankenstein finetuning notebooks.
- `nn-architectures-project1/` - neural network architecture notebooks.
- `nn-architectures-project2/` - CLIP/autoencoder notebooks.
- `recommendation-system-performance/` - recommendation system analysis notebooks.
