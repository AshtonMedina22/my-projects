LangChain app project

Sequential quiz generation with focused AI chains.

A LangChain application that turns a topic into a quiz question, then passes that generated question into a second chain for a detailed answer and explanation.

Problem

Educational content generation is repetitive but quality-sensitive.

This project breaks quiz generation into two auditable steps: question creation and answer explanation. The smaller chain responsibilities make the workflow easier to test, improve, and later expand into more advanced educational content tools.

Architecture

SequentialChain Data Flow

01

PromptTemplate: formats topic, difficulty, and audience into a controlled question-generation prompt.

02

Question LLMChain: uses Gemini to generate one clear learner-facing quiz question.

03

Answer LLMChain: receives the generated question and returns an answer, explanation, and key takeaway.

04

SequentialChain: passes the question output into the answer chain automatically through output keys.

App features

Portfolio-Ready Controls

Generation

Topic, level, and audience controls

The app lets users tune quiz generation for different learners instead of hardcoding one generic prompt.

Operations

Demo mode without exposing secrets

Public reviewers can inspect the interface without API keys, while the real Gemini path runs when environment variables are configured.

Evidence

History and export

Generated quizzes are stored in session history and can be exported as CSV for review or downstream content workflows.
