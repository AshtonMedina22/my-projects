# Using Session State for App Persistence

Codecademy AI Engineer lesson reference for making the Streamlit sentiment app remember text analysis history with `st.session_state`.

## Files

- `sentiment_app.py` - paste-ready cumulative Streamlit app for the lesson checkpoints.
- `LESSON_NOTES.md` - checkpoint requirements and solution snippets.

## Run

```bash
streamlit run sentiment_app.py
```
