# Lesson Notes

## What This Lesson Does

This lesson introduces the basic Streamlit elements needed to start a sentiment analysis dashboard.

## Main Streamlit Elements

- `st.markdown()` displays formatted Markdown text.
- `with st.sidebar:` creates a sidebar area.
- `st.header()` creates a section heading.
- `st.subheader()` creates a smaller section heading.
- `st.dataframe()` displays an interactive table.
- `st.line_chart()` displays a simple interactive line chart.

## Important Behavior

Streamlit apps run from top to bottom. When the user changes an input, Streamlit reruns the script and updates the UI.
