# Using Python in Streamlit

Codecademy lesson: **Build AI Applications with Streamlit - Using Python in Streamlit**

This lesson starts a Streamlit sentiment analysis dashboard using `sentiment_app.py`.

## Files

- `sentiment_app.py` - paste-ready app script for the lesson checkpoints.
- `LESSON_NOTES.md` - plain-English notes for later reference.

## Run Command

```bash
streamlit run sentiment_app.py
```

In the Codecademy workspace, refresh:

```text
http://localhost:4001/
```
