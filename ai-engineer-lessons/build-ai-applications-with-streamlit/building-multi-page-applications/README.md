# Building Multi-Page Applications

Codecademy AI Engineer lesson reference for organizing a Streamlit sentiment analysis app with tabs.

## Files

- `sentiment_app.py` - completed paste-ready Codecademy lesson file.
- `LESSON_NOTES.md` - checkpoint summary and exact required tab structure.

## Checkpoints

1. Create three tabs with `st.tabs(["📝 Text Input", "📁 File Upload", "📊 Results & Analytics"])`.
2. Move CSV batch upload and analysis into the file upload tab.
3. Move analysis history, metrics, charts, and history table into the results tab.
