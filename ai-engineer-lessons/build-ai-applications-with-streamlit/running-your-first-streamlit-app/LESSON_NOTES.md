# Lesson Notes

## Streamlit Purpose

Streamlit is a Python library for quickly turning Python scripts into interactive web apps.

## Key Command

```bash
streamlit hello
```

## Important Streamlit Behaviors

- Streamlit starts a local web server.
- In Codecademy, the app is shown through the embedded browser, usually at `localhost:4001`.
- When a user changes an input, Streamlit reruns the script from top to bottom.
- Streamlit provides built-in UI components for tables, charts, text, sidebars, and controls.

## Demo Sections To Explore
