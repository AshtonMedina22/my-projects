# Running Your First Streamlit App

Codecademy lesson: **Build AI Applications with Streamlit - Running Your First Streamlit App**

This lesson introduces Streamlit by running its built-in demo app.

## Command

```bash
streamlit hello
```

In the Codecademy workspace, the browser should load the app at:

```text
http://localhost:4001/
```

## What To Notice

- Streamlit turns Python scripts into interactive web apps.
- UI inputs trigger automatic reruns from top to bottom.
- The sidebar organizes different demo sections.
- Data tables and plots are interactive without custom HTML/CSS/JavaScript.
