# User Input and App State

Codecademy AI Engineer lesson reference for adding text input, button-triggered analysis, CSV upload, and a simple list memory demonstration to the Streamlit sentiment app.

## Files

- `sentiment_app.py` - paste-ready cumulative Streamlit app for the lesson checkpoints.
- `LESSON_NOTES.md` - checkpoint requirements and solution snippets.

## Run

```bash
streamlit run sentiment_app.py
```
