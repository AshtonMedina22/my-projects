# Build AI Applications with Streamlit

This folder stores Codecademy AI Engineer lesson work for building interactive Python apps with Streamlit.

## Lessons

- `running-your-first-streamlit-app/` - first Streamlit demo using `streamlit hello`.
- `using-python-in-streamlit/` - first custom Streamlit app for a sentiment analysis dashboard.
- `buttons-sliders-and-other-ui-components/` - Streamlit widget lesson adding selectbox, sliders, and a load-model button.
- `displaying-data-and-visualizations/` - Streamlit lesson adding KPI metrics, charts, and interactive data tables.
- `user-input-and-app-state/` - Streamlit lesson adding text input, CSV upload, and a simple app-state demonstration.
- `using-session-state-for-app-persistence/` - Streamlit lesson using `st.session_state` to persist text analysis history.
- `building-multi-page-applications/` - Streamlit lesson organizing the sentiment analysis app into tabs.
- `connecting-to-databases/` - Streamlit lesson saving and loading sentiment analysis results with SQLite.
- `loading-and-running-models/` - Streamlit lesson loading Hugging Face sentiment models and running predictions.
- `interactive-ml-applications/` - Streamlit lesson comparing preprocessing choices and model predictions.
- `deploying-ai-applications-to-streamlit-community-cloud/` - Streamlit deployment notes for GitHub and Community Cloud.
