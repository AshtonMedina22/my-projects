# Best Practices in AI Deployment

Codecademy AI Engineer course section for deploying AI systems responsibly.

## Lessons

- `data-hygiene-and-privacy-basics/` - Validate support data, redact PII, hash customer IDs, and apply basic differential privacy to statistics.
- `monitoring-101/` - Calculate latency, quality, cost, and weekly trend metrics for chatbot operations.
- `evaluations-that-matter/` - Compare model versions with offline metrics, significance tests, and online A/B results.
