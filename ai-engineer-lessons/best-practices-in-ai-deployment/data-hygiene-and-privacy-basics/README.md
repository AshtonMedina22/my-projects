# Data Hygiene & Privacy Basics

Codecademy AI Engineer lesson for cleaning customer support data before AI deployment.

## Lesson Goal

Validate raw customer support records, remove obvious PII from text, anonymize customer identifiers with hashing, and apply basic differential privacy to grouped ZIP code statistics.

## Files

- `lesson_text.txt` - Exported lesson prompt text.
- `notebook.md` - Exported notebook text.
- `checkpoint_solutions.py` - Paste-ready checkpoint code blocks.
- `LESSON_NOTES.md` - Short operational notes for the lesson.
