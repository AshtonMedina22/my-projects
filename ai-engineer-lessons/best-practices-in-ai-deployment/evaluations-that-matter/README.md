# Evaluations That Matter

Codecademy AI Engineer lesson for comparing model versions before deployment.

## Lesson Goal

Run offline evaluation metrics, test whether accuracy differences are statistically significant, then analyze online A/B test results for resolution rate and cost.

## Files

- `notebook.md` - Exported notebook text.
- `checkpoint_solutions.py` - Paste-ready checkpoint code blocks.
- `LESSON_NOTES.md` - Short operational notes for the lesson.
