# Lesson Notes

This lesson compares two chatbot model versions using offline and online evaluation.

Key steps:

- Calculate offline accuracy and hallucination rates for Model A and Model B.
- Use a Chi-squared test to determine whether accuracy differences are statistically significant.
- Split A/B production results into control and treatment groups.
- Compare resolution rates and average costs.
- Use a second Chi-squared test to evaluate whether online resolution differences are statistically significant.

This is a notebook-only deployment best-practices lesson. It does not add a Streamlit app feature.
