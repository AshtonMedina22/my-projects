# Lesson Notes

This lesson introduces basic monitoring for deployed AI chatbot systems.

Key metrics:

- Average, p95, and p99 latency.
- Slow conversation count and percentage.
- Resolution rate, hallucination rate, CSAT, and escalation rate.
- Total cost, average cost, token usage, and yearly cost projection.
- Weekly trend analysis for resolution, CSAT, and latency.

This is a notebook-only deployment best-practices lesson. It does not add a Streamlit app feature.
