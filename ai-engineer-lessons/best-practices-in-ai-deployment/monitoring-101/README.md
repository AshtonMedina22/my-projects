# Monitoring 101

Codecademy AI Engineer lesson for calculating chatbot monitoring metrics.

## Lesson Goal

Measure technical performance, quality, cost, and weekly trends from production-style chatbot conversation logs.

## Files

- `lesson_text.txt` - Exported lesson prompt text.
- `notebook.md` - Exported notebook text.
- `checkpoint_solutions.py` - Paste-ready checkpoint code blocks.
- `LESSON_NOTES.md` - Short operational notes for the lesson.
