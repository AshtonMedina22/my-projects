# Assignment Brief

## Building an Agentic Meal Planner

This project combines the agentic techniques from the course into one workflow.

The user provides:

- number of people
- number of days
- daily calorie target
- food allergens
- budget in USD
- pantry groceries already available

The workflow:

1. Drafts a JSON meal plan.
2. Critiques the draft for constraint violations and improvement suggestions.
3. Revises the plan through repeated evaluation/refinement passes.
4. Builds a grocery list after the final plan is ready.

## Concepts

- Prompt chaining
- JSON mode
- Orchestration
- Evaluation
- Iterative refinement
