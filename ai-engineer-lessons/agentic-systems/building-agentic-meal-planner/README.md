# Building an Agentic Meal Planner

Course lesson/project notes for the AI Engineer agentic workflow project.

## Project Goal

Build an agentic meal planner that:

1. Accepts user constraints for people, days, daily calories, allergens, budget, and pantry items.
2. Drafts a weekly dinner plan in JSON mode.
3. Critiques the plan for cost, calorie, allergy, and pantry-use constraints.
4. Revises the plan through repeated critique/refinement passes.
5. Builds a grocery list that excludes pantry items already available.

## Techniques Practiced

- Prompt chaining
- JSON mode
- Orchestration
- Evaluation/refinement loops
- Agentic workflow control
