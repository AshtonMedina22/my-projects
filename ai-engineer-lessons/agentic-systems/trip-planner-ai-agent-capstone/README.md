# Trip Planner AI Agent Capstone

This folder tracks the AI Engineer capstone project for building a Streamlit trip planner agent with OpenAI tool calling, OpenStreetMap data, optional Wikivoyage RAG, PyDeck maps, persistence, refinement, and feedback loops.

Portfolio implementation:

`../../../trip-planner-ai-agent/`

## Core Concepts

- OpenAI Responses API
- Function calling and tool orchestration
- Nominatim geocoding
- Overpass POI search
- Wikivoyage RAG
- Streamlit session state
- PyDeck map rendering
- JSON persistence and feedback loops
