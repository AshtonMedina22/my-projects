# Implementing Evaluation Metrics

Codecademy lesson: **Evaluating LLMs - Implementing Evaluation Metrics**

This lesson covers two common language model evaluation metrics:

- **Perplexity**: Measures how surprised a causal language model is by a text sequence. Lower is better.
- **BLEU**: Measures n-gram overlap between generated text and reference text. Higher is better.

## Files

- `checkpoint_solutions.py` - paste-ready checkpoint answers.
- `LESSON_NOTES.md` - plain-English notes for later portfolio/project reference.

## Key Takeaway

Perplexity is useful for checking how likely a language model thinks text is. BLEU is useful for checking how closely generated text matches a reference answer.
