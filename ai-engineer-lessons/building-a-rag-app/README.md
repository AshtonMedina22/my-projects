# Building a RAG App

Codecademy AI Engineer course section for turning RAG prototypes into an interactive Streamlit application.

## Lessons

- `streamlit-introduction/` - Streamlit UI basics for a RAG demo app.
- `uploading-documents/` - Persistent Chroma setup, anthropology document chunking, metadata upload, and sample queries.
- `similarity-search-in-streamlit/` - Streamlit app that queries the persistent Chroma `RAG_Assistant` collection.
- `prompt-engineering-in-streamlit/` - Streamlit RAG app lesson adding OpenAI completions and metadata-aware citation prompts.
