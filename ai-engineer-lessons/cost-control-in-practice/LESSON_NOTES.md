# Lesson Notes

## Main Concepts

- Cost control routes easy questions to cheaper systems before using expensive models.
- Tier 1 can use rule-based answers for common FAQs.
- Tier 2 can use a cheaper model for normal customer-service questions.
- Tier 3 should be reserved for complex, sensitive, or failed cases.
- Caching repeated questions saves money because only the first occurrence needs an API call.
- Cost savings must be evaluated together with quality metrics such as resolution rate and CSAT.

## Checkpoint Summary

1. Calculate tier distribution, current cost, all-GPT-4 baseline cost, and routing savings.
2. Hash repeated questions, identify duplicates, estimate cache hit rate, and calculate cache savings.
3. Group by tier to compare resolution rate and average CSAT.
