# Contextual Query Rewriting

Codecademy AI Engineer lesson for improving RAG retrieval by rewriting context-dependent user questions into standalone search queries.

## Lesson Goal

Use chat history to clarify the latest query before running vector search, then send the retrieved results through the existing RAG prompt and completion helpers.

## Files

- `notebook.md` - Exported lesson notebook text.
- `checkpoint_solutions.py` - Paste-ready checkpoint code blocks.
- `LESSON_NOTES.md` - Short operational notes for the lesson.
