# Hypothetical Document Embeddings

Codecademy AI Engineer lesson for implementing HyDE retrieval in a RAG workflow.

## Lesson Goal

Generate a hypothetical answer from the user query, use that generated text for vector retrieval, then feed the retrieved source chunks into the normal RAG prompt.

## Files

- `notebook.md` - Exported lesson notebook text.
- `checkpoint_solutions.py` - Paste-ready checkpoint code blocks.
- `LESSON_NOTES.md` - Short operational notes for the lesson.
