# Decoupling

Codecademy AI Engineer lesson for improving RAG retrieval by decoupling the initially retrieved chunk from the context sent to the generation model.

## Lesson Goal

Use the chunk index from an initial Chroma search result to retrieve the previous and next chunks, then build a larger RAG prompt containing the neighboring context.

## Files

- `notebook.md` - Exported lesson notebook text.
- `checkpoint_solutions.py` - Paste-ready checkpoint code blocks.
- `LESSON_NOTES.md` - Short operational notes for the lesson.
