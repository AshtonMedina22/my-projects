# Fusion Search

Codecademy AI Engineer lesson for decomposing complex RAG questions into smaller subquestions.

## Lesson Goal

Ask the model to generate JSON-formatted subquestions, answer each subquestion through the RAG pipeline, then synthesize the subquestion answers into one final answer.

## Files

- `notebook.md` - Exported lesson notebook text.
- `checkpoint_solutions.py` - Paste-ready checkpoint code blocks.
- `LESSON_NOTES.md` - Short operational notes for the lesson.
