# Evaluating LLMs on Q&A Tasks

Codecademy lesson: **LLM Benchmarks - Evaluating LLMs on Q&A tasks**

This lesson evaluates a DistilBERT question-answering model on SQuAD using Exact Match and F1 score.

## Files

- `checkpoint_solutions.py` - paste-ready checkpoint answers.
- `LESSON_NOTES.md` - plain-English notes for later reference.
- `notebook.md` - exported lesson notebook/source.

## Key Takeaway

Q&A evaluation checks whether the model can extract the correct answer span from context. Exact Match rewards perfectly matching answers, while F1 gives partial credit for overlapping answer tokens.
