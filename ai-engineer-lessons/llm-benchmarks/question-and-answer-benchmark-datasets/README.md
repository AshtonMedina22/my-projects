# Question and Answer Benchmark Datasets

Codecademy lesson: **LLM Benchmarks - Question and Answer Benchmark Datasets**

This lesson uses SQuAD with a DistilBERT question-answering model to generate extractive answers from context passages.

## Files

- `checkpoint_solutions.py` - paste-ready checkpoint answers.
- `LESSON_NOTES.md` - plain-English notes for later reference.
- `notebook.md` - exported lesson notebook/source.

## Key Takeaway

Question-answering benchmarks like SQuAD test whether a model can read a context passage and extract the answer span that best responds to a natural-language question.
