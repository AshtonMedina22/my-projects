# Sentiment Analysis Benchmark Datasets

Codecademy lesson: **LLM Benchmarks - Sentiment Analysis Benchmark Datasets**

This lesson uses the SST-2 sentiment dataset from the GLUE benchmark and evaluates a pretrained DistilBERT sentiment classifier.

## Files

- `checkpoint_solutions.py` - paste-ready checkpoint answers.
- `LESSON_NOTES.md` - plain-English lesson notes.
- `notebook.md` - exported lesson notebook/source.

## Key Takeaway

Benchmark datasets like SST-2 make model evaluation more standardized because models are tested on the same task, data format, and metrics.
