# LLM Benchmarks

This folder stores Codecademy AI Engineer lesson work for benchmark datasets and model evaluation.

## Lessons

- `sentiment-analysis-benchmark-datasets/` - SST-2 sentiment benchmark evaluation with DistilBERT.
- `evaluating-sentiment-analysis/` - GLUE SST-2 evaluation with Hugging Face `Trainer` and accuracy metrics.
- `question-and-answer-benchmark-datasets/` - SQuAD question-answering benchmark lesson with DistilBERT.
- `evaluating-llms-on-qa-tasks/` - SQuAD Q&A evaluation using Exact Match and F1 metrics.
- `general-knowledge-benchmark-datasets/` - MMLU general knowledge benchmark lesson with FLAN-T5.
- `benchmarking-llms-on-multiple-nlp-tasks/` - project benchmarking DistilBERT on SST-2 and SQuAD.
