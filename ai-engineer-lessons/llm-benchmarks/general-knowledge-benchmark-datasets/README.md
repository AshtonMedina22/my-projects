# General Knowledge Benchmark Datasets

Codecademy lesson: **LLM Benchmarks - General Knowledge Benchmark Datasets**

This lesson evaluates `google/flan-t5-small` on a small sample from the MMLU benchmark.

## Files

- `checkpoint_solutions.py` - paste-ready checkpoint answers.
- `LESSON_NOTES.md` - plain-English notes for later reference.
- `notebook.md` - exported lesson notebook/source.

## Key Takeaway

General knowledge benchmarks like MMLU test whether a model can answer questions across many subject areas. Smaller models may struggle, especially on specialized topics.
