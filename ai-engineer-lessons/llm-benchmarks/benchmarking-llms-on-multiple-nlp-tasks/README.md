# Benchmarking LLMs on Multiple NLP Tasks

Codecademy project: **Benchmarking LLMs on Multiple NLP Tasks**

This project benchmarks DistilBERT on two standard NLP tasks:

- Sentiment analysis with SST-2 from GLUE.
- Extractive question answering with SQuAD.

## Files

- `task_solutions.py` - paste-ready task answers.
- `LESSON_NOTES.md` - plain-English notes for later reference.
- `notebook.md` - exported project notebook/source.

## Key Takeaway

Benchmarking across multiple tasks shows that one model family can be evaluated differently depending on the task: classification uses accuracy, precision, and recall, while question answering uses exact match and F1-style answer comparison.
