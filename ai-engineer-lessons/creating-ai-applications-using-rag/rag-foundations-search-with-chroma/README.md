# RAG Foundations: Search With Chroma

Codecademy lesson for using Chroma as a vector database for semantic search.

## Files

- `checkpoint_solutions.py` - paste-ready checkpoint answers.
- `LESSON_NOTES.md` - short reference notes and source links.
- `notebook.md` - exported lesson notebook/source from the assignment.
