# Prompt Engineering for RAG

Codecademy lesson for building RAG prompts from Chroma search results and generating completions with the OpenAI API.

## Files

- `checkpoint_solutions.py` - paste-ready checkpoint answers.
- `LESSON_NOTES.md` - short reference notes for later review.
- `notebook.md` - exported lesson notebook/source from the assignment.
