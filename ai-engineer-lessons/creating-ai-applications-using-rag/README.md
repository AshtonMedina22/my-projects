# Creating AI Applications Using RAG

Codecademy AI Engineer course section for retrieval-augmented generation foundations.

## Lessons

- `rag-foundations-search-with-chroma/` - Chroma vector database lesson covering embeddings, collections, and semantic search.
- `rag-foundations-document-chunking-with-rag/` - RAG foundations lesson covering document chunking, overlap, metadata, and Chroma uploads.
- `prompt-engineering-for-rag/` - RAG lesson building prompts from Chroma search results and adding source-citation instructions.
