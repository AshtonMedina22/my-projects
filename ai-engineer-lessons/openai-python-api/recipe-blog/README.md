# OpenAI Python API: Recipe Blog

## Lesson Purpose

This lesson practices using the OpenAI Python API to generate recipe recommendations for a food blog.

The project asks us to provide the model with context about:

- Dietary preferences
- Ingredients the user already has
- Favorite cuisines
- Desired blog-friendly recipe formatting

## What This Demonstrates

- Prompt engineering for structured recipe output.
- Using user context to tailor AI-generated content.
- Preparing model output for a real content use case, such as a recipe blog.
- Building API-based Python scripts that can later become portfolio tools.

## Files

- `script.py` - lesson code that we will update as checkpoints are assigned.
- `LESSON_NOTES.md` - task instructions and completed checkpoint notes.

## Portfolio Relevance

This lesson can later become a polished portfolio project: an AI recipe assistant that creates blog-ready recipes from pantry ingredients and dietary preferences.
