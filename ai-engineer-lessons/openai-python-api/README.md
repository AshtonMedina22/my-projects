# OpenAI Python API Lessons

This section contains Codecademy AI Engineer lessons focused on using the OpenAI Python API.

Each lesson folder should include:

- `script.py` - working lesson code or checkpoint-ready snippets.
- `LESSON_NOTES.md` - task instructions, decisions, and final answers.
- `README.md` - short description of the lesson and what skill it demonstrates.

Current lessons:

- `recipe-blog/` - recipe generator for a food blog using user preferences, ingredients, and cuisine context.
